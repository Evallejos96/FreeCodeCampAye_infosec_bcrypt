# FCC Advanced Node and Express

## Overview
A freeCodeCamp boilerplate project for learning Advanced Node.js and Express concepts, specifically focused on bcrypt for password hashing.

## Project Structure
- `server.js` - Main Express server entry point
- `freeCodeCamp/fcctesting.js` - FCC testing utilities (do not edit)
- `package.json` - Node.js dependencies and scripts

## Running the Project
The server runs on port 5000 and is configured via the `npm start` command which runs `node server.js`.

## Dependencies
- express - Web framework
- bcrypt - Password hashing
- body-parser - Request body parsing
- cors - Cross-origin resource sharing
- helmet - Security headers
- mongodb - MongoDB driver

## Configuration
- Port: 5000 (configurable via PORT environment variable)
- Server binds to 0.0.0.0 for Replit compatibility
